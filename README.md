# 🎧 Spotify Clone

Landing page inspirada na interface do Spotify, desenvolvida do absoluto zero com foco em prática, aprendizado e evolução como desenvolvedor.

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

## 📄 Sobre o projeto

Este projeto é um clone do Spotify criado com o objetivo de reproduzir elementos visuais e interações da plataforma original. Mais do que o resultado final, o foco foi o processo: construir cada detalhe do zero, entendendo na prática como estruturar layouts, estilizar componentes e adicionar interatividade com JavaScript.

Desenvolver esse projeto foi desafiador, pois exigiu atenção aos detalhes, organização do código e resolução de problemas reais que surgem ao longo do desenvolvimento. Mesmo sendo um projeto simples, ele fortalece muito a base e ajuda a evoluir como programador.
